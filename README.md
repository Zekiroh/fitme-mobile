# FitMe Mobile Application

Android fitness application built with Kotlin as part of a full-stack gym member management capstone system.

## Overview

FitMe Mobile acts as a fitness companion application for gym members. It allows users to track workouts, monitor progress, apply for memberships, and communicate with coaches.

The application connects to the FitMe backend API.

## Features

- Login and registration
- Initial fitness assessment
- Workout tracking
- Progress monitoring
- Membership application
- Customer assistance messaging
- Attendance visibility (if integrated)

## Built With

- Kotlin
- Android Studio
- Retrofit
- REST API integration

## Configuration

Update API base URL inside the Retrofit client configuration:

`BASE_URL = "http://your-server-url/`

## Running the App

1. Open project in Android Studio
2. Sync Gradle
3. Run on emulator or physical device
